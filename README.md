# Library App

## Demo 
https://joshua-2333.github.io/Library/

A dynamic library management web application that allows users to store, organize, and track books. Built with JavaScript using object-oriented principles and DOM manipulation for a fully interactive experience.

---

## Features

- Add new books (title, author, pages, read status)
- Remove books from the library
- Toggle read/unread status
- Uses object constructors / classes
- Instant UI updates without page reload
- Responsive and user-friendly interface

---

## Built With

- HTML5  
- CSS3  
- JavaScript (ES6+)  

---

## Overview

This project simulates a personal digital library where users can manage their book collection. It demonstrates:

- Dynamic rendering of book cards  
- Event-driven UI updates  
- Object-oriented JavaScript design  
- Clean and organized component structure
